## Intelligent Street Lighting System

This project was completed as part of my B.Tech curriculum and focuses on designing an energy-efficient and sustainable street lighting solution.

### Overview
The system uses solar energy, proximity sensing, and automatic light control to reduce power consumption while maintaining effective street illumination.

### Key Concepts
- Intelligent lighting based on vehicle and pedestrian movement
- Solar-powered system with tracking mechanism
- Energy optimization and sustainability

### Tools & Technologies
Arduino, Sensors (LDR, RF modules), Embedded Systems, Solar Power Concepts
